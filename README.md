# keep-going

A tiny command-line morale booster. Prints “Keep Going!” and exits successfully.

Project repo: [keep-going](https://github.com/djsoftware1/keep-going)

## Usage

```sh
$ keep-going
$ keep-going --random
```

A good addition to your .bashrc or motd.

For extra fun, combine with cowsay (optional):

```sh
$ keep-going | cowsay

$ keep-going -r | cowsay
```
